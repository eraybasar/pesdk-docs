title: Which browsers are supported?
---

WebGL Renderer:

  * IE 11+
  * FF 15+
  * Chrome 11+
  * Safari 5.1+
  * Opera 19+
  * iOS Safari 8+
  * Firefox for Android 45+
  * Android Browser 47+
  * Opera Mobile 36+

Canvas Renderer (slower than WebGL):

  * IE 9+
  * FF 10+
  * Chrome 11+
  * Safari 2.0+
  * Opera 12+
  * iOS Safari 3.2+
  * Firefox for Android 45+
  * Android Browser 3+
  * Opera Mobile 12+

Regarding webcam support, please consult [caniuse.com](http://caniuse.com/#feat=stream).
