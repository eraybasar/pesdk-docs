getValueAsInts

open fun getValueAsInts(): IntArray!

Gets the value as an array of ints. This method should be used for tags of type #TYPE_UNSIGNED_SHORT, #TYPE_UNSIGNED_LONG.

Return
IntArray!: the value as as an array of ints, or null if the tag's value does not exist or cannot be converted to an array of ints.
