MaskedCropAspectAsset(id: String!, cropWidth: Int, cropHeight: Int, fixedSize: Boolean, exportMask: Boolean)

Creates a new instance, of a aspect ration, which is also masking the image by a given color.

Parameters

id - String!: a unique identifier for this asset.

cropWidth - Int: width of the aspect.

cropHeight - Int: height of the aspect.

fixedSize - Boolean: if true the cropWidth and cropHeight are interpreted as Pixel values and force the image to export at this specific resolution.

exportMask - Boolean: it true, the mask is applied to the exported image, otherwise it is only visual for the user while editing.
