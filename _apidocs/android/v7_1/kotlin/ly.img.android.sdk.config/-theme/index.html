Properties

background

 var background: Color?

The background color of the canvas. If null it defaults to the ExistingTheme.DARK's value or the value of the theme that is modified.

menuBackground

 var menuBackground: Color?

The background color of the menu and accesory controls above the menu. If null it defaults to the ExistingTheme.DARK's value or the value of the theme that is modified.

primary

 var primary: Color?

The primary color which is mainly used for the text and the icons on the menu bar. If null it defaults to the ExistingTheme.DARK's value or the value of the theme that is modified.

tint

 var tint: Color?

The tint color of highlighted user interface elements. If null it defaults to the system tint color if available or the value of the theme that is modified.

toolbarBackground

 var toolbarBackground: Color?

The background color of the toolbar that hosts the title of the active tool, the discard, and the apply button. If null it defaults to the ExistingTheme.DARK's value or the value of the theme that is modified.
