minimumSize

 var minimumSize: Float?

The minimum size that a brush can have. This value is measured in relation to the smaller side of the image that the user is editing. If the value is null the minimum brush size is set to the absolute size of a single pixel of the edited image. The default value is 0 and will mean, 1 Pixel at the minimum.
