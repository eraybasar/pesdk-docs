SCENE_CAPTURE_TYPE

SCENE_CAPTURE_TYPE

Value is int. This tag indicates the type of scene that was shot. It can also be used to record the mode in which the image was shot. Note that this differs from the scene type (SceneType) tag.

  • 0 = Standard
  • 1 = Landscape
  • 2 = Portrait
  • 3 = Night scene
  • Other = reserved
