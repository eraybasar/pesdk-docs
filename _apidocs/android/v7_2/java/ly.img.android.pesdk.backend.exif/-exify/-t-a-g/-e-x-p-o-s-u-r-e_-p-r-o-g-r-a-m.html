EXPOSURE_PROGRAM

EXPOSURE_PROGRAM

Value is unsigned int. Exposure program that the camera used when image was taken.

  • '1' means manual control
  • '2' program normal
  • '3' aperture priority
  • '4' shutter priority
  • '5' program creative (slow program)
  • '6' program action(high-speed program)
  • '7' portrait mode
  • '8' landscape mode.
