class URI

A URI for the look up table (LUT) image.

A URI for the thumbnail image of the category. If null the category will not have a thumbnail image which won't be required if flattenCategories is enabled for the Tool.TRANSFORM.

A URI for the end image. If null there is no end image.

A URI for the middle image.

A URI for the start image. If null there is no start image.

A URI for the thumbnail image of the frame.

A URI for the overlay image.

A URI for the thumbnail image of the overlay. If null the thumbnail will be automatically generated form the overlayURI.

A URI for the sticker image.

A URI for the thumbnail image of the sticker. If null the thumbnail will be automatically generated from the stickerURI.

A URI for the thumbnail image of the category.

A URI for the font file. TTF and OTF fonts are allowed. If null the fontName is assumed to be a system font.
