onBeforeDetach

protected open fun onBeforeDetach(panelView: View!, revertChanges: Boolean): Int

Called before the View will detached. It is possible that this method will not call. Use #onDetached() instated to clear critical allocations.

Return
Int: a delay in ms. This will give time to Animate.
